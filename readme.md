# AstrBot TTS语音插件

对接GPT-SoVITS的文本转语音插件。

## 功能特性

- 支持将文本转换为语音消息
- 支持缓存机制，避免重复请求
- 支持自定义音色和语种
- 集成GPT-SoVITS API

## 前置条件

1. 安装并启动GPT-SoVITS API服务

2. 准备参考音频文件（时长建议3-10秒）和对应的文本内容

3. 配置AstrBot机器人，确保平台可以正确触发大模型回复消息，插件会使用配置的第一个大模型机器人

## 配置说明

在AstrBot插件配置中设置以下参数：

- **tts_enabled**: 是否启用TTS功能（默认：false）
- **tts_api_url**: TTS API地址（默认：http://127.0.0.1:9880）
- **ref_audio_path**: GPT-SoVITS参考音频路径（必填,默认为目录下自带的参考音频）
- **prompt_text**: 参考音频对应的文本内容（必填，默认为目录下自带的参考音频对应的文本内容）

## 使用方法

### 语音命令

将文本直接转换为语音。
示例：
```
@机器人 语音 你好
```

令机器人并使用语音回复大模型的消息。
示例：
```
@机器人 语音回复 你好
```

### 语音回复命令

先获取大模型的回复，然后将回复转换为语音发送。

示例：
```
@机器人 语音回复 今天天气怎么样
```

这将先让大模型回答"今天天气怎么样"，然后将回答内容转换为语音发送。

### 注意事项

1. 首次使用前需要在插件配置中启用TTS功能并设置相关参数
2. 参考音频路径必须是服务器本地可访问的路径
3. 转换结果会缓存7天，相同文本和音色会直接使用缓存
